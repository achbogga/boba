# Boba

Boba is building an advisory-first control plane for Ray Data and KubeRay
operators who need fewer scheduling surprises, lower spill risk, and better
heterogeneous cluster utilization.

The initial prototype focuses on:

- preflight validation
- schedulability analysis
- spill and memory risk detection
- cluster-shape recommendations
- replayable decision traces

For the evidence-backed product thesis and contributor call, start with
[Why Boba Matters: A Call For Core Contributors](why-boba-matters/index.md).
