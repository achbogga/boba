# KubeRay demo environments

This directory contains the reproducible demo assets for the Boba prototype.

## Profiles

- `kind`: local CPU-only smoke path for contributor onboarding
- `gpu`: heterogeneous node-pool profile for external GPU-capable Kubernetes
  clusters

## Versions

- Ray `2.54.0`
- KubeRay operator `1.5.1`

These versions align with the current Ray and KubeRay documentation used by the
prototype.
